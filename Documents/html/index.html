Analog Devices Sensor Software Reference Manual

Introduction

This product provides software for many Analog Devices sensors such as gas sensors, temperature sensors, accelerometers, and visable light sensors. The software is targeted at Analog Devices' low-power Cortex-M microcontrollers, such as the ADuCM3029.

API Reference Manual

This "Analog Devices Sensor Software Reference Manual" documents a C++ programming environment providing an Application Programming Interface (API), supporting a quick and easy functional interface to Analog Devices sensors.

The manual describes the sensor class heirarchy and the different functions the classes implement.
